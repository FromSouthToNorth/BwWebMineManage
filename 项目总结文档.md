## 1. 项目概述

Bw.WebMineManage 是一个面向煤矿行业的**矿山安全管理前端系统**，提供安全监测、人员定位、矿压监测、产量监控等核心功能。系统同时支持 PC 端和移动端访问，由外部系统通过 SSO 集成跳转使用。

- **项目名称**：`bw-webminemanage`
- **项目类型**：前端 SPA（单页应用）
- **开发语言**：JavaScript (Vue 2) + TypeScript (迁移中)
- **包管理器**：Yarn（备选 pnpm）
- **部署路径**：`/WebMineManage`（子路径部署）
- **Docker 端口映射**：`33373:80`

## 5. 架构设计

### 5.1 路由设计

所有路由均设置 `hidden: true`，因为应用不提供侧边栏导航，而是由外部系统通过 URL 直接访问。路由模式为 **history 模式**（无 `#` 号）。

- PC 页：标准路由路径，如 `/index.cpt`、`/personnelLocation.cpt`
- 移动端：以 `_phone.cpt` 后缀标识，如 `/index_phone.cpt`

### 5.2 认证与 SSO

应用不维护独立登录页，通过 **单点登录（SSO）** 集成：

1. **SSO SDK 方式**：依赖全局 `BW_SSO_SDK` 对象自动登录
2. **URL 参数方式**：通过 `mine_key` 参数（Base64 编码的 `用户名;密码`）直接登录
3. **Token 传递**：支持 URL 参数 `token` 传递认证令牌
4. **Token 存储**：通过 Cookie + localStorage 双重存储（`cookie.js` 自定义实现）

### 5.3 API 调用模式

所有后端接口通过 **Strategy 接口** 统一调用，使用固定的 `{ id, parameter }` 结构：

```javascript
// 三种调用方式：
// 1. 普通数据（GetStrategyData）
// 2. 分页数据（GetPageStrategyData）
// 3. JSON 格式数据（GetStrategyJsonData）
// 4. 执行命令（ExecuteStrategyCom）
```

接口基地址通过环境变量 `VUE_APP_BASE_API` 配置，开发环境默认代理到 `http://192.163.5.10:33382/`。

### 5.4 Vuex Store

7 个模块：
- **app** — 侧边栏、移动端检测、Element UI 尺寸、弹窗状态
- **user** — Token、用户名
- **settings** — 主题
- **safetyMonitoring** — 安全监测数据、矿井名称
- **production** — 产量数据（昨日/今日、近年趋势、瞬时流量）
- **minePressure** — 矿压数据（安装位置、类型、报警值）
- **personnelLocation** — 人员定位（报警数据、下井人数、表格、弹窗、屏幕信息）

### 5.5 主题系统

支持暗黑主题（`theme-dark`）和浅色主题（`theme-light`）：

- 默认主题在 `src/settings.js` 中配置为 `theme-dark`
- 可通过 URL 参数 `data-theme` 切换
- 主题写入 `localStorage` 持久化
- 切换后设置到 `<html>` 标签属性，CSS 通过属性选择器响应
- CSS 变量定义在 `root.scss` / `root.css` 中

### 5.6 全局挂载

`src/main.js` 中全局注册：

- **方法**：`parseTime`、`resetForm`、`addDateRange`
- **消息**：`msgSuccess`、`msgError`、`msgWarn`、`msgInfo`
- **组件**：`Pagination`、`SafetyMonitoringStatistics`、`RightToolbar`、`TopShow`、`Setting`

### 5.7 屏幕适配

- 监听 `orientationchange` 事件检测横竖屏
- 根据 userAgent 判断移动端/PC
- 根据窗口高度动态调整 Element UI 尺寸（`small`/`middle`/`default`）
