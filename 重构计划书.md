## 一、概述

### 1.1 项目背景

原项目是一个面向煤矿行业的矿山安全管理前端系统，提供安全监测、人员定位、矿压监测、产量监控等核心功能。同时支持 PC 端（Element UI）和移动端（Vant 2），由外部系统通过 SSO 集成跳转使用。

### 1.2 重构目标

| 维度 | 源项目 | 目标 |
|------|--------|------|
| 框架 | Vue 2.6.12 | Vue 3.x |
| 构建工具 | Vue CLI 4 (webpack) | Vite 5 |
| 语言 | JavaScript | TypeScript |
| 包管理器 | Yarn | pnpm |
| PC UI | Element UI 2.15 | Element Plus |
| 移动端 UI | Vant 2.13 | Vant 4 |
| 状态管理 | Vuex 3 | Pinia 2 |
| 测试 | 无 | Playwright E2E |
| 样式 | 18个 SCSS 文件 | **全新设计** |
| 代码风格 | Options API | `<script setup lang="ts">` |

### 1.3 核心原则

1. **不碰原项目** — 在 `D:\bw\BwWebMineManage` 从零构建，原项目完整保留
2. **功能对等** — 所有路由、页面功能、API 接口与原项目一致
3. **样式全新** — 不迁移旧样式，重新设计现代暗色主题
4. **测试先行** — Playwright E2E 测试覆盖核心路径
5. **组件复用逻辑提取** — Mixins → Composables，大型组件拆分
